# vim 命令

  • :set nu,显示行号
  • :set nonu,取消显示行号
  • :n,定位到第n行
  • :wq 储存后离开，若为 :wq! 则为强制储存后离开
  • :q 离开 vim
  • :w 将编辑的数据写入硬盘档案中
  • :! command 暂时离开 vim 到指令行模式下执行 command 的显示结果！例如:! ls /home即可在 vim 当中察看 /home 底下以 ls 输出的档案信息！
